# 🤖 NareshIT Bot using LangChain, Ollama & DeepSeek-R1

A simple AI chatbot built using **LangChain**, **Ollama**, **DeepSeek-R1**, and **Streamlit**. The chatbot runs locally using Ollama and provides intelligent responses to user queries through a clean web interface.

---

## 🚀 Features

- 💬 Interactive AI Chatbot
- 🧠 Powered by LangChain
- ⚡ Local LLM using Ollama
- 🤖 Supports DeepSeek-R1 (or any Ollama model)
- 🎨 User-friendly Streamlit Interface
- 🔒 Runs completely on your local machine

---

## 🛠️ Tech Stack

- Python 3.12
- Streamlit
- LangChain
- Ollama
- DeepSeek-R1
- LangChain Core

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/deepseek-chatbot.git
cd deepseek-chatbot
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Virtual Environment

**Windows**

```bash
.venv\Scripts\activate
```

**Linux / macOS**

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Install Ollama

Download Ollama from:

https://ollama.com/download

---

## Pull DeepSeek Model

```bash
ollama pull deepseek-r1
```

Or use another installed model such as:

```bash
ollama pull qwen2.5:3b
```

---

## ▶️ Run the Application

```bash
streamlit run deepseek_app.py
```

The application will open automatically in your browser.
